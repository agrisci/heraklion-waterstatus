# Welcome

Heraklion Waterstatus was born by the fact that the city i live in has often unexpected water cut-offs. This can lead at broken appliances e.g washing machine, dishwasher or consuming water while there is no supply for not critical tasks.

In the need of an application that will inform me of water cut-offs as soon as they happen and the thrill of learning and practising Angular at the same time i wrote this app.

# Features

#### Implemented:

- Create different users
- Subscribe users to multiple areas
- Enable/Disable Notifications
- Support Email Notifications
- Support Discord Notifications
- Application Logs

#### Not implemented yet:

- Limit Backend Requests (caching)
- Add Login/Registration Page
- Add Gotify Notifications
- Maybe add support for different cities

### Overall a major UI rewrite will be done in the future
